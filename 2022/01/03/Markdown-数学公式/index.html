Markdown-数学公式

1.基本语法

1.1 行内公式

  • 行内公式的格式是$...$
  • 例如$g(z) = {1\over 1+e^{-z}}$的写法是：$g(z) = {1\over 1+e^{-z}}$

1.2 整行公式

  • 整行公式的格式是$$...$$
  • 例如
    $$h_\theta(x) = g(\theta^Tx)$$
  • 写法是：$$h_\theta(x) = g(\theta^Tx)$$

3.上标与下标

  • 上标用^来表示，例如$x^2$表示为：$x^2$
  • 下标用-来表示，例如$x_2$表示为：$x_2$
  • 若上标或下标是一个表达式，则需要加上{}，例如$x^{t-1}$表示为：$x^{t-1}$
